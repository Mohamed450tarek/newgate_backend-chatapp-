A real-time chat application built with **React**, **Node.js**, **Express**, **MongoDB**, and **Socket.io**. This app allows users to sign up, sign in, and chat with other users in real-time. It supports private messaging, notifications, and a responsive UI.

## Features

- User authentication (Sign up & Sign in)
- Real-time messaging using **Socket.io**
- Group and private chats
- Online/offline status of users
- Responsive design for mobile and desktop
- Notifications for new messages
- Secure password storage with hashing
- JWT-based authentication for API requests

## Tech Stack

- **Frontend:** React, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js, Socket.io
- **Database:** MongoDB (Atlas)
- **Authentication:** JWT, bcrypt
- **Deployment:** Vercel (Frontend), Railway/Heroku (Backend)
